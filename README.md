# jetsnack_flutter

Clon de la app Jetsnack, implementado en Flutter. Presenta un catálogo de snacks con navegación moderna, UI responsiva y patrones comunes en apps móviles.

## Qué incluye
- Lista de productos, búsqueda y filtrado básicos
- Pantalla de detalles del snack
- Carrito de compras y flujo de pago simulado
- Theming y soporte para iOS y Android

## Objetivo
Proyecto educativo para practicar Flutter/Dart replicando la experiencia de Jetsnack en el ecosistema Flutter.
